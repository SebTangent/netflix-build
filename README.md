# Sebflix - Netflix Clone

## Introduction

Sebflix is a Netflix clone created with React, Stripe, and Cloudflare.

## Features

- User authentication
- Movie and TV show browsing
- Search functionality
- Stripe for Monthly payments
- Responsive design

## Getting Started

### Coding Languages / APIs / Etc

- Node.js
- Redux
- TMBD API
- Stripe 
- Cloudflare
