# Sebflix - Netflix Clone

## Introduction

Sebflix is a Netflix clone created with React, Stripe, and Cloudflare.

## Features

- User authentication
- Movie and TV show browsing
- Search functionality
- Stripe for payments
- Responsive design

## Getting Started

### Prerequisites

- Stripe account
